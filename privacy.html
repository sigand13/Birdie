Vilken information samlar vi in?

Vi samlar inte in några personuppgifter direkt via webbplatsen (vi använder inga cookies eller formulär). När du kontaktar oss via telefon eller SMS sparar vi dina kontaktuppgifter endast för att kunna genomföra affären.

Hur används informationen?

Informationen används endast för att kommunicera med dig angående ditt köp av golfbollar.

Tredjepart

Vi delar aldrig dina uppgifter med tredje part. Webbplatsen hostas på GitHub Pages, som kan samla in loggfiler för säkerhetsändamål.
